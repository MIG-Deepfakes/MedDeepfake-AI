🧠 Medical Deepfake Detection in Brain MRI

Fourier Analysis + Deep Learning for Robust Detection

📖 Abstract

Recent advances in generative models have enabled the creation of highly realistic synthetic medical images. While useful for data augmentation, these images pose serious risks such as misdiagnosis. This project presents a framework for detecting fake brain MRI images using handcrafted features (ELA, wavelet, gradient, Fourier) along with deep feature embeddings from VGG19. The framework is evaluated using SVM and CNN models and achieves up to 99.8% accuracy, with Fourier features proving most effective in identifying subtle frequency-domain inconsistencies.
